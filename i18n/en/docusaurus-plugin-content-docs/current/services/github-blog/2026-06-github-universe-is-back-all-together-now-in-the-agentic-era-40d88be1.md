# GitHub Universe is back: All together now, in the agentic era

> GitHub Blog · 2026-06-04 · Company news

원문 링크: [GitHub Universe is back: All together now, in the agentic era](https://github.blog/news-insights/company-news/github-universe-is-back-all-together-now-in-the-agentic-era/)

GitHub이 연례 콘퍼런스 GitHub Universe를 2026년 10월 28–29일 샌프란시스코의 Fort Mason Center로 돌아온다고 발표했습니다. 행사 제목은 "All together now, in the agentic era"입니다.
제공된 정보만 보면 이번 행사는 대면 중심의 복귀와 함께 ‘agentic era’라는 테마를 내세우는 것으로 보입니다. 자세한 일정과 등록 정보는 원문 확인이 필요합니다.

## 핵심 포인트

- GitHub Universe가 2026년 10월 28–29일에 개최됩니다.
- 장소는 샌프란시스코 Fort Mason Center입니다.
- 행사 테마는 "agentic era"로 표기되어 있습니다.

## 읽어볼 만한 이유

피드 기준으로는 GitHub의 대면 행사 복귀와 AI/에이전트 관련 논의 가능성을 시사합니다. 개발자와 조직의 로드맵·생산성 도구 변화에 영향을 미칠 수 있습니다.
