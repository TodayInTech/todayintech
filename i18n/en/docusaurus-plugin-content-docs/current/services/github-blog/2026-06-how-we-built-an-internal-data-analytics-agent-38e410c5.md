# How we built an internal data analytics agent

> GitHub Blog · 2026-06-19 · AI &amp; ML

원문 링크: [How we built an internal data analytics agent](https://github.blog/ai-and-ml/github-copilot/how-we-built-an-internal-data-analytics-agent/)

---

GitHub 내부에서 개발한 데이터 분석 에이전트 'Qubot'을 소개하는 글입니다. Qubot은 Copilot 기반으로 동작하며, 모든 GitHub 직원이 자연어로 데이터에 대해 질문할 수 있게 설계되었다고 피드 요약에 나와 있습니다.
작성자는 구축 과정에서 얻은 교훈을 공유한다고 합니다. 제공된 정보만 보면 기술적 구현이나 성능 수치 등 상세 내용은 포함되어 있지 않습니다.

## 핵심 포인트

- Qubot은 내부용 Copilot 기반 데이터 분석 에이전트입니다.
- 직원 누구나 자연어로 데이터 질의가 가능하도록 고안되었습니다.
- 글은 구축 과정에서의 학습점과 경험을 공유하는 내용입니다.

## 읽어볼 만한 이유

기업 내부에서 Copilot을 분석 에이전트로 활용한 사례라는 점에서, 유사한 내부 도구 도입이나 AI 에이전트 설계에 참고가 될 수 있습니다.

## 확인할 점

- 피드 기준으로는 구현 세부사항과 성능 지표가 제공되지 않습니다.
