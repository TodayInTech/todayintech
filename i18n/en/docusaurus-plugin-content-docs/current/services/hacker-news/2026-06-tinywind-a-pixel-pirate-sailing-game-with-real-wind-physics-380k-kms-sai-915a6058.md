# TinyWind: A pixel pirate sailing game with real wind physics (380k+ kms sailed)

TinyWind은 픽셀 그래픽의 해적 항해 게임입니다. 제공된 정보만 보면 ‘실제 풍력 물리’를 적용했다는 점과 총 항해 거리 380k+ km 같은 메트릭이 눈에 띕니다.

- 픽셀 그래픽의 해적 항해 게임.

- Hacker News에서 큰 반응(포인트 545, 댓글 111) — tinywind.io에서 플레이 가능.

## 읽어볼 만한 이유

웹에서 물리 기반 시뮬레이션을 구현한 사례라는 점에서 개발자와 시뮬레이션 관심자에게 흥미롭습니다. 커뮤니티 반응으로 실제 사용성과 아이디어의 잠재력을 가늠해볼 수 있습니다.
