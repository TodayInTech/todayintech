# The text in Claude Code’s “Extended Thinking” output

---

제목에 따르면 Claude Code의 'Extended Thinking' 기능에서 생성된 텍스트가 진정성이 없다는 주장이 제기됩니다. 피드 기준으로는 원문에서 어떤 근거와 예시로 그런 결론을 내렸는지, 저자가 어떤 방법으로 분석했는지는 확인할 수 없습니다. 제공된 메타데이터에는 작성자(0o_MrPatrick_o0)와 원문 URL만 포함되어 있어 상세한 재현성·증거는 원문을 직접 확인해야 합니다.
