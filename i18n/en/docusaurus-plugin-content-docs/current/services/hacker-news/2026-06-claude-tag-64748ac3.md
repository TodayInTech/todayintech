# Claude Tag

> Hacker News · 2026-06-23 · 인공지능

---

Anthropic은 팀 단위 협업을 위해 설계된 새로운 방식의 Claude, 'Claude Tag'를 공개했다. 클라우드 애플리케이션이 아닌 Slack 내 팀 구성원으로 참여해 선택된 채널과 연동 도구·데이터·코드베이스에 접근하도록 설정할 수 있으며, 사용자가 채널에서 @Claude를 태그하면 모델이 해당 요청을 단계로 분해해 순차적으로 수행하고 결과를 스레드로 보고한다. 한 채널에 하나의 Claude가 돌아가는 '멀티플레이어' 설계로서 누구나 진행 상태를 확인하고 이어받을 수 있으며, 채널을 따라 쌓인 문맥을 기억해 반복 설명을 줄여주고(사적 채널은 제외) 권한이 허용되면 다른 채널과 데이터 소스에서 자동으로 학습하기도 한다. Anthropic 내부에서는 제품팀 코드의 65%가 내부 Claude Tag로 생성되는 등 업무 효율 개선 사례를 언급한다.
관리자 중심의 접근 제어와 가시성 기능도 핵심이다. 관리자는 Claude가 접근할 도구와 정보를 채널 별로 엄격히 지정해 각 사용 사례별로 별도 '정체성'을 만들 수 있고, 메모리도 채널 범위에 국한되어 교차 노출을 막는다. 조직·채널 단위 토큰 사용 한도 설정과 활동 로그(누가 어떤 작업을 요청했는지)도 제공되며 기존 Slack 앱을 대체, 30일 내 마이그레이션 선택권과 일부 고객 대상 도입 크레딧이 있다. 비동기적 작업 예약과 스스로 과제를 추진하는 능력, 채널 단위의 공동작업 특성은 다수 작업을 병렬로 위임하는 워크플로를 현실화한다는 점에서 엔터프라이즈 AI 에이전트 적용 사례로서 기술적·운영적 의미가 크다. Claude Tag는 Opus 4.8로 동작하며 현재 Claude Enterprise 및 Team 고객 대상 베타로 제공된다.
