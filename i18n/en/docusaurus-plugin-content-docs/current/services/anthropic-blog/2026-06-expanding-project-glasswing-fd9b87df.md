# Expanding Project Glasswing

피드 기준으로는 Anthropic가 Project Glasswing을 약 150개의 신규 기관으로 확장한다고 발표했습니다. 확장 대상은 15개국 이상이라고만 요약되어 있습니다.
제공된 정보만 보면 확장 규모만 공개된 상태입니다. 참여 기관 명단, 목표나 일정 같은 상세 내용은 원문에서 확인해야 합니다.

## 핵심 포인트

- Anthropic이 Project Glasswing 확대를 발표함.
- 약 150개의 신규 기관을 추가한다는 내용.
- 대상은 15개국 이상으로 표기됨 (게시일 2026-06-02, Anthropic Blog).

## 읽어볼 만한 이유

피드 기준으로는 지리적·조직적 범위 확대로 프로젝트의 시범·연구 범위가 커질 가능성을 시사합니다. 관련 생태계 영향이나 협력 확대 여부를 가늠할 단서가 될 수 있습니다.

## 확인할 점

- 피드 기준으로는 구체적 기관 명단, 참여 조건, 일정 등 세부 정보가 제공되지 않음.
