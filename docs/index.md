# Today in Tech

Today in Tech는 AI 기반 기술 뉴스 브리핑 플랫폼입니다.

브리핑은 날짜별 디렉터리 아래에 생성됩니다. 각 날짜는 전체 요약 문서와 서비스별 문서로 구성됩니다.

```text
docs/YYYY-MM-DD/
├── summary.md
└── services/
    ├── hacker-news.md
    ├── github-blog.md
    ├── google-blog.md
    ├── openai-blog.md
    └── anthropic-blog.md
```
