# Microsoft Comic Chat is now open source

> Hacker News · 2026-07-16 · 오픈소스·소프트웨어 역사

---

마이크로소프트가 1990년대 IRC 대화를 자동으로 만화 패널로 변환해 널리 알려진 ‘Comic Chat’의 소스 코드를 오픈소스로 공개했다. 이 클라이언트는 대화의 문맥을 해석해 등장인물의 표정, 제스처, 말풍선과 패널 레이아웃을 실시간으로 결정하는 방식으로 단순한 스킨을 넘어선 자동 일러스트 제작·편집 기능을 담고 있었다. Comic Sans 서체가 실제로 처음 쓰이기 시작한 환경 중 하나이며, Jim Woodring의 일러스트와 DJ Kurlander, Tim Skelly, David Salesin 등이 기여한 기술적·예술적 결합이 주목할 만하다.
원문은 Comic Chat이 Visual C++ 4.0과 MFC로 개발되어 1996년 Internet Explorer 3 번들로 배포되었고, SIGGRAPH ’96에서 기술을 정리한 점을 근거로 그 기술적 의미를 짚는다. 오픈소스 공개물에는 원본 스냅샷과 함께 현대 Visual Studio로의 빌드, 현대 IRC 서버 연결, 고해상도 환경에서의 실행을 보여주는 AI 기반의 모더나이즈 예제들이 포함되어 있어 연구·학습·포팅·재창조의 출발점이 된다. 보존 가치와 동시에 과거 실험적 아이디어가 오늘날 메시징 기능(이모티콘, 아바타, 자동 표현 선택 등)에 미친 영향을 돌아보게 한다.
