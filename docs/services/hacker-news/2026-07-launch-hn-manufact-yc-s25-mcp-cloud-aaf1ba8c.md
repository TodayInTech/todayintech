# Launch HN: Manufact (YC S25) – MCP Cloud

> Hacker News · 2026-07-02 · Product Launch / Cloud Platform

---

Manufact는 MCP(모델 컨텍스트 프로토콜) 앱과 서버를 위한 수직형 클라우드로서, 창업자들이 이전에 운영하던 오픈소스 SDK(mcp-use)를 바탕으로 개발자 경험과 배포 파이프라인을 통합하는 것을 목표로 한다. 사이트와 소개 자료는 호스팅, 브랜치별 프리뷰, GitHub 앱을 통한 자동 배포, 커스텀 도메인·SSL 지원 같은 기본적인 배포 기능 외에도 크로스-클라이언트 테스트(예: ChatGPT, Claude, Gemini 대상), 퍼블리싱 체크리스트와 제출용 자산 자동 생성, 임베디드 퍼블릭 채팅 등 MCP 앱을 스토어에 올리고 운영하기 위한 실무적 도구들을 강조한다. Cloud Inspector는 실환경 트래픽을 추적·재현하고 JSON-RPC를 검사하며 모델 교체와 자동 평가(evals)를 통해 다양한 클라이언트·모델에서의 동작을 확인하게 해 준다.
제품 설명은 MCP 생태계에서 에이전트 하네스(harness)들이 표준화되는 변화를 배경으로 삼아, 서버 측 통합의 중요성이 커졌다는 관점을 반영한다. npx create-mcp-use-app으로 스캐폴딩을 제공하고, SDK 기반의 단일 코드베이스에서 여러 클라이언트에 배포할 수 있도록 한 점은 개발·테스트·모니터링·스토어 제출까지의 흐름을 줄여 실무 부담을 낮추려는 설계 의도를 보여준다. 이러한 기능들은 MCP가 표준으로 자리잡고 주요 호스트들이 마켓플레이스를 운영하는 환경에서, 기업 내부 시스템과의 일체화 및 크로스-클라이언트 호환성 확보라는 기술적 요구를 충족시키는 실용적 해법으로 읽힌다.
