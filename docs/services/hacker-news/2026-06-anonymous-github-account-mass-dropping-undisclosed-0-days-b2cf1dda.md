# Anonymous GitHub account mass-dropping undisclosed 0-days

> Hacker News · 2026-06-27 · 보안

---

익명 GitHub 계정이 공개되지 않은 제로데이(0-day) 취약점을 대량으로 공개하고 있다는 제목의 소식이 Hacker News에서 큰 주목을 받고 있다. 피드 메타데이터에 따르면 관련 저장소는 https://github.com/bikini/exploitarium 이며 게시자는 'binyu'로 표기되어 있고, Hacker News에서 590점과 232개의 댓글이 집계되었다. 다만 제공된 정보만 보면 원문 본문이나 취약점의 세부 내용은 포함되어 있지 않다.
기술적 의미로는 미공개 제로데이의 대량 공개 가능성이 탐지·대응·패치 배포 등 보안 운영의 우선순위를 급격히 바꿀 수 있다는 점에서 보안팀과 오픈소스 유지관리자가 주의를 기울여야 한다. 그러나 피드 메타데이터만으로는 영향을 받는 프로젝트 범위, 악용 증거, 공개 방식(증명 코드 포함 여부) 등 핵심 정보가 확인되지 않아 구체적 영향 평가와 대응 권고를 위해서는 원문과 추가 자료 검토가 필수적이다.
