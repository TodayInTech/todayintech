# Celebrating 25 years of visual search innovation

> Google Blog · 2026-07-14 · Search / AI

---

Google이 Google Images 출시 25주년을 맞아 시각 검색 경험을 확장하는 두 가지 주요 업데이트를 발표했습니다. 첫째는 데스크톱(미국, 영어)에서 몇 주 내 순차 도입될 '브라우저블 홈'으로, 웹 전역의 이미지를 실시간으로 갤러리 형태로 보여주고 사용자가 컬렉션에 저장한 아이디어를 탭 형태로 노출해 탐색 흐름을 유지하도록 설계되었습니다. 둘째는 검색 내 AI Overviews에 이미지 생성을 직접 통합한 기능으로, Google의 최신 이미지 생성 모델인 Nano Banana를 사용해 텍스트 프롬프트로부터 고품질 이미지를 새로 만들어냅니다. 이미지 생성은 영어로, 현재 AI Mode에서 이미지 생성 지원 지역에 순차 도입될 예정입니다. 사용자 계정 로그인으로 기능을 체험할 수 있습니다.
이 발표는 지난 25년간의 기술적 이력을 배경으로 멀티모달 검색이 어떻게 진화했는지를 요약합니다. 2001년 이미지 검색 출범 이후 유사 이미지, Search by Image, Lens 통합, 멀티서치, Circle to Search, Lens+AI Mode의 결합 등을 거치며 시각 정보가 단순 조회를 넘어 장면 전체를 해석하고 하위 쿼리로 분해하는 'visual image fan-out' 같은 기법으로 확장됐습니다. 2025년의 Search Live와 2026년의 Circle to Search 다중 객체 인식, 그리고 이번의 지능형 검색 상자는 여러 이미지를 업로드해 상세한 질문을 할 수 있게 하며, 검색이 보는 것과 상상하는 것을 하나로 잇는 방향으로 나아가고 있음을 보여줍니다. 기술적으로는 이미지 생성과 고급 시각 해석을 검색 환경에 직접 결합함으로써 창작과 탐색 간 경계가 좁아지고, 멀티모달 입력을 통한 문맥화·추론 능력이 더욱 중요해졌다는 점이 핵심적입니다.
