# Claude for Creative Work

> Anthropic Blog · 2026-07-13 · AI / Creative Tools

---

Anthropic은 창작 작업에서 Claude의 활용을 넓히기 위해 주요 크리에이티브 소프트웨어와 직접 연결하는 커넥터들을 공개했다. 회사는 Claude가 창의력이나 취향을 대체하지는 못하지만 아이디어 발굴을 가속하고 반복적 노동을 줄이며 더 큰 규모의 프로젝트를 다룰 수 있게 해준다고 설명한다. 공개된 항목에는 Ableton(제품 문서 기반 응답), Adobe Creative Cloud(Photoshop·Premiere 등 50개 이상 앱 연동), Affinity by Canva(배치 작업·파일 내보내기 자동화), Autodesk Fusion(대화로 3D 모델 생성·수정), Blender(파이썬 API 자연어 인터페이스), Resolume Arena/ Wire(실시간 라이브 시각 제어), SketchUp(대화로 3D 모델 시작점 생성), Splice(샘플 카탈로그 검색) 등이 포함되어 있다. 또한 Claude Code를 통한 스크립트·플러그인 자동 생성, 형식 변환으로 툴 간 파이프라인 연결, 배치 처리 같은 반복 업무 자동화와 Claude Design을 통한 시각화·Canva로의 결과 내보내기 등 실무적 사용 사례도 제시한다.
Blender와 관련해서는 Blender 개발자들이 만든 MCP 커넥터를 통해 Claude가 씬 분석·디버그, 일괄 스크립트 적용, Blender 인터페이스에 도구 추가까지 수행할 수 있으며, Anthropic이 Blender에 일회성 기부를 했고 MCP 기반이므로 다른 LLM들도 접근 가능하다고 명시했다. 교육 분야 협력도 발표돼 RISD, Ringling, Goldsmiths의 일부 프로그램에 Claude와 커넥터 접근권을 제공하고 현장 피드백을 수집할 계획이다. 전반적으로 이번 발표는 대형 언어 모델을 기존 창작 소프트웨어의 API·파이프라인과 직접 연결해 코드 생성, 자연어 인터페이스, 자산 동기화 같은 기능을 통해 창작 워크플로우의 자동화와 상호운용성을 높이려는 기술적 방향을 분명히 보여준다. Updated May 1, 2026 관련 문구도 기부 처리 방식 변경(Blender Development Fund 대신 일회성 기부)으로 수정되었음을 밝히고 있다.
