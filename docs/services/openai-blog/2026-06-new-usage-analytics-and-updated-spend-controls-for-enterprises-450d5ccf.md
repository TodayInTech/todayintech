# New usage analytics and updated spend controls for enterprises

> OpenAI Blog · 2026-06-18 · Product

원문 링크: [New usage analytics and updated spend controls for enterprises](https://openai.com/index/chatgpt-enterprise-spend-controls)

---

피드 기준으로는 OpenAI가 ChatGPT Enterprise용 새로운 사용량 분석과 지출 통제 기능을 발표했습니다. 제공된 정보만 보면 이 업데이트는 조직이 AI 사용량을 모니터링하고 비용을 관리하도록 돕는 데 초점을 둡니다.
기업 고객 관점에서 비용 통제와 사용 분석을 함께 제공함으로써 AI 도입을 보다 안정적으로 확장할 수 있다는 점을 강조합니다. 세부 기능과 도입 방식은 원문에서 추가 확인이 필요합니다.

## 핵심 포인트

- ChatGPT Enterprise에 사용량 분석 기능이 추가됨.
- 기업용 지출 통제(spend controls) 기능이 도입됨.
- 목표는 조직의 비용 관리와 AI 확장 지원.

## 읽어볼 만한 이유

피드 기준으로는 기업이 AI 사용 증가에 따른 비용과 거버넌스를 관리해야 하는 상황에서 관련 도구는 운영·예산 리스크를 줄이는 데 유용할 가능성이 큽니다.

## 확인할 점

- 피드 기준으로는 기능 세부사항·가격·출시 시점이 명시되어 있지 않음.
- 원문에서 API 적용 범위·권한 제어 등 구체 조건을 확인할 필요가 있음.
