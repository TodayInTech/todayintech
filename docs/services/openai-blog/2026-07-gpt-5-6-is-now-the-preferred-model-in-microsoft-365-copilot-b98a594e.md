# GPT-5.6 is now the preferred model in Microsoft 365 Copilot

> OpenAI Blog · 2026-07-09 · Product

---

피드 메타데이터에 따르면 OpenAI는 GPT-5.6을 Microsoft 365 Copilot의 선호 모델로 지정했다고 전합니다. 제공된 설명은 Word, Excel, PowerPoint, Chat, Cowork 전반에서 GPT-5.6이 더 강력한 AI 역량을 발휘해 작업을 더 빠르고 높은 품질로 돕는다는 점을 강조합니다. 원문 전문은 포함돼 있지 않아 구체적 사례나 수치, 적용 방식은 확인되지 않습니다.
이 변화는 최신 대형언어모델이 사무용 생산성 도구 핵심에 통합되는 사례로서 의미가 있습니다. 피드 정보만으로는 응답 품질 개선의 정도, 지연 시간·비용·배포 전략 같은 기술적 상세는 알기 어렵지만, 엔터프라이즈 관리자와 개발자 관점에서는 향후 릴리스 노트와 실제 적용 사례를 통해 성능·통합 방식의 구체적 영향을 점검할 필요가 있습니다.
