# $100 million for open source: A milestone built by the community

> GitHub Blog · 2026-07-20 · Open Source Funding

---

GitHub가 2019년 시작한 GitHub Sponsors 프로그램이 커뮤니티와 기업 후원을 통해 누적 1억 달러를 투자했다는 발표는 단순한 금액을 넘어 오픈소스 유지보수 생태계의 자금 흐름이 바뀌고 있음을 시사한다. 프로그램은 개인 후원으로 시작해 2023년 조직 후원을 일반 제공하며 기업 수준의 대규모 후원을 가능하게 했고, 103개 지역으로 확장되며 Patreon과의 파트너십, 청구서 결제와 대량 후원 같은 마찰을 줄이는 도구들을 도입했다. 그 결과 7만개 이상의 유지보수자와 조직이 지원을 받고, 28만 건이 넘는 스폰서 참여가 이뤄졌으며 자금 유입 속도도 빨라져 초기 1천만 달러 도달에 가까운 2년이 걸렸던 반면 최근 1천만 달러는 단 5개월 만에 달성됐다. 여러 유지보수자와 기업(예: Shopify, Mercedes‑Benz)의 사례 인용은 후원이 실제로 개인의 전업 전환, 프로젝트 장기 투자, 기업의 기술 의존성 해소 수단으로 기능하고 있음을 보여준다.
게시자는 이 경험에서 얻은 교훈들을 기술 독자 관점에서 구체적으로 제시한다. 조직 후원이 개인 기여보다 규모 면에서 유의미한 효과를 내고 있고, 후원 과정의 마찰을 줄일 때 자금이 증가한다는 점은 기업이나 플랫폼 설계자가 고려할 실무적 인사이트다. 동시에 보고서는 여전히 큰 자금 격차와 유지보수자 번아웃이 소프트웨어 공급망의 주요 위험으로 남아 있음을 경고하며, 이를 해소하기 위한 현실적 행동으로 '의존하는 프로젝트 후원'과 조직 차원의 참여를 권장한다. 후원 참여를 원하는 유지보수자와 사용자에게는 Sponsors Explore 페이지와 github.com/sponsors를 통해 프로필 설정과 후원 시작을 권하는 실무적 안내가 포함되어 있어 즉각적 행동으로 연결하기 쉬운 내용이다.
