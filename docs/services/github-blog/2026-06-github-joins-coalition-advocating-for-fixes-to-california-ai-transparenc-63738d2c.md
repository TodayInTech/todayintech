# GitHub joins coalition advocating for fixes to California AI Transparency Act to protect open source

> GitHub Blog · 2026-06-23 · 정책

---

GitHub이 Black Forest Labs, Hugging Face, Mozilla와 함께 캘리포니아 AI 투명성 법안(SB 942, SB 1000 수정안)에 대해 오픈소스 생태계와 충돌하는 조항을 수정할 것을 요구하는 연대 서한에 참여했다. 핵심 쟁점은 법안 초안의 ‘라이선스 취소’ 규정으로, 오픈소스 라이선스는 본질적으로 영구적이고 취소할 수 없다는 원칙에 의해 작동한다. 그러나 초안은 하류 사용자가 특정 의무를 이행하지 않을 경우 라이선스를 취소하도록 요구하는데, 이런 방식은 널리 사용되는 오픈소스 관행과 맞지 않아 협업·커뮤니티 기반 프로젝트와 전체 소프트웨어 공급망에 불확실성을 초래할 수 있다고 연대가 지적한다.
연대 서한은 이 규정이 법의 투명성 목표를 달성하는 데 필수적이지 않다고 보고, 이미 수정·배포하는 개발자는 법의 직접 적용 대상이며 집행 수단도 존재한다고 설명한다. 대안으로는 EU의 AI법 투명성 실무 코드(Transparency Code of Practice)처럼 오픈소스의 특성을 인정하고 하류 사용자에게 문서화 등 권고를 통지하는 접근을 권장한다. GitHub은 이러한 수정이 법의 투명성 목적을 유지하면서 오픈소스 개발과의 호환성을 보장해 AI 책임성과 개방적 혁신을 동시에 지킬 수 있다고 보고 있으며, 기술적으로 근거 있는 피드백이 정책입안자에게 전달되길 촉구하고 있다.
