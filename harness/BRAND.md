# Today in Tech 브랜드

Today in Tech의 기본 브랜드 마크는 `T` 모노그램과 우상향 신호를 결합한 형태다.

- `T`: Today와 Tech
- 단순한 `T` 모노그램: 작은 크기에서도 유지되는 Today와 Tech의 정체성
- 청록색 점: 오늘 새롭게 포착된 핵심 신호

## 자산

- 라이트 모드: `static/img/brand/todayintech.light.svg`
- 다크 모드: `static/img/brand/todayintech.dark.svg`
- 조직 아이콘: `static/img/brand/todayintech.organization.svg`

라이트/다크 자산은 파비콘, 내비게이션 로고, 메인 사이드바 항목에 사용한다. 조직 아이콘 자산은 GitHub 조직 avatar처럼 테마 전환이 없는 정사각형 표면에 사용한다. 비율이나 내부 요소를 변형하지 않는다.
