# Today in Tech 브랜드

Today in Tech의 기본 브랜드 마크는 `T` 모노그램과 우상향 신호를 결합한 형태다.

- `T`: Today와 Tech
- 단순한 `T` 모노그램: 작은 크기에서도 유지되는 Today와 Tech의 정체성
- 청록색 점: 오늘 새롭게 포착된 핵심 신호

## 자산

- 라이트 모드: `static/img/brand/todayintech.light.svg`
- 다크 모드: `static/img/brand/todayintech.dark.svg`
- 조직 아이콘: `static/img/brand/todayintech.organization.svg`
- 업로드용 PNG:
  - `static/img/brand/todayintech.light.png`
  - `static/img/brand/todayintech.dark.png`
  - `static/img/brand/todayintech.organization.png`

SVG는 원본 벡터, 사이트 UI, README 표시용으로 사용한다. PNG는 GitHub 조직 avatar처럼 PNG 업로드가 필요한 정사각형 표면에 사용한다. 비율이나 내부 요소를 변형하지 않는다.
